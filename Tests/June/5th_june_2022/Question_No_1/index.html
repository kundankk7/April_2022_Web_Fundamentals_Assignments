Question 1:

In how many types can we create functions? State the difference between those types by giving examples.

There can be 4 different types of user-defined functions, they are:

1. Functions without arguments and without return values


This function has no arguments and no return value:


#include
int main()
{
void sum();
sum();
}
void sum() //function with no arguments and return data type
{
int x=18,y=38,z=8;
printf("x = %d ; y = %d ; z = %d \n",x,y,z);
int sum = x+y+z;
printf("Sum = %d",sum);
}



2. Functions without arguments and with return values


This function has no arguments, but it has a return value:


#include
int main()
{
int sum();
int c = sum();
printf("Sum = %d",c);
}
int sum() //function with no arguments and return data type
{
int x=27,y=56,z=9;
printf("x = %d ; y = %d ; z = %d \n",x,y,z);
int sum = x+y+z;
return(sum);
}



3. Functions with arguments and without return values


This function has arguments, but it does not return a value


#include
int main()
{
void sum(float,float); //function with arguments and no return value
float x=90.96,y=56.56;
sum(x,y);
}
void sum(float a,float b) //function with arguments and no return value
{
float z = a+b;
printf("x + y = %f",z);
}



4. Functions with arguments and return values


This function has arguments and returns a value:


#include
void main()
{
int sub(int,int); //function with return value and arguments
int x=24,y=4;
int res = sub(x,y);
printf("x-y = %d",res);
}
int sub(int a,int b) //function with return value and arguments
{
return(a-b); // return value
}
