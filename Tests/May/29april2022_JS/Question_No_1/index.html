Q 1.Explain difference between var/ let const by giving examples.

Var:- Variables declared using var keyword scoped to the current execution context. This means If they are inside a function we only can access them inside the function. and If they are not they are part of the global scope which we can access anywhere. look at the example below to better understanding.

Let:- et creates variables that can not be redeclared and can be updated.

Const:- Const variables are cannot be updated or redeclared.
